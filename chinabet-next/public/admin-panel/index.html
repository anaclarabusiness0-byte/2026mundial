Pagamentos via Asaas

🔑 Chave de API do Asaas

Acesse app.asaas.com → Integrações para obter sua chave. Use Sandbox para testes e Produção quando for ao ar.

📋 Como configurar

1. Crie sua conta em asaas.com
2. Acesse Configurações → Integrações → API Key
3. Copie a chave e cole aqui
4. Configure o Webhook no painel Asaas para a URL:
https://SEU_DOMINIO/api/webhooks/asaas
5. Eventos a ativar: PAYMENT_RECEIVED e PAYMENT_CONFIRMED
✅ Com o webhook configurado, o saldo é creditado automaticamente quando o pagamento é confirmado.
